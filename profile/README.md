# Avid Media Composer

**Avid Media Composer** is the professional non-linear editing system trusted by Hollywood studios, broadcast networks, and independent filmmakers worldwide. As a powerful **Avid Media Access Tool**, you can link directly to camera originals, QuickTime, MXF, and image sequences without importing or copying media. This **Avid Media Indexer** automatically creates searchable metadata, PhraseFind audio indexing, and ScriptSync for documentary and narrative workflows. With **Avid Media Database**, the system manages media on shared storage with robust bin locking, project sharing, and background consolidation.
